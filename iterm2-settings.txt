2b. <Option 2> : 18 pt Anonymice Nerd Font Complete Mono
Font (under tab 'text') :

Anonymice Nerd Font Complete Mono: 
 - set -g theme_nerd_fonts yes  #(type that in the terminal) (this is for bobTheFish omf theme for fish shell)
 - downlad the .ttf file here https://github.com/ryanoasis/nerd-fonts/blob/master/patched-fonts/AnonymousPro/complete/Anonymice%20Nerd%20Font%20Complete%20Mono.ttf
 - font size is 18pt (mentioned in "18 pt Anonymice Nerd...")

 - change the font style in iterm-2 preferences>profiles>text>Change font> choose "Anonymice Nerd Font Complete Mono"
